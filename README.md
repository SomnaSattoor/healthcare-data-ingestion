# Healthcare Eligibility Pipeline

## How to Run
1. Install Python 3
2. Install dependency:
   pip install pandas
3. Ensure acme.txt and bettercare.csv are in the project folder
4. Run:
   python pipeline.py

## Output
Creates unified_output.csv with standardized schema.

## Adding a New Partner
Add a new configuration entry in config.py.
No changes required in pipeline.py.
